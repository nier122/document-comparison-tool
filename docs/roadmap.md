# Product Roadmap

## Phase 1 - MVP
- Upload two PDFs
- Extract text from generated PDFs
- Compare content
- Show side-by-side PDFs
- Highlight differences
- Difference navigation panel

Current implementation focus:
- Generated PDFs only
- Frontend-only comparison workflow
- No OCR
- No backend services

## Phase 2 - Extraction Quality
- Improve coordinate-based text extraction
- Header and footer detection
- Column-aware reading order
- Table-aware extraction
- Better document structure extraction
- Search within compared documents

## Phase 3 - OCR Support
- Scanned PDF OCR
- OCR quality review and correction workflows
- Preserve generated-PDF comparison as the primary stable path

## Phase 4 - Advanced OCR and Review
- Handwritten PDF OCR as a future advanced feature
- Standards database
- Clause comparison
- AI review comments
- Compliance checking

## Phase 5 - Collaboration and Enterprise
- Team collaboration
- Audit logs
- Version history
- Enterprise workflows
