# Codex Prompt

You are the lead software engineer for this repository.

Before making decisions, read docs/project_context.md.

Your goal is to build Version 1 (MVP) of the Document Comparison Tool.

Requirements:

1. Use React and TypeScript for the frontend.
2. Use Node.js and Express for the backend.
3. Allow upload of two PDF documents.
4. Extract text from generated PDFs.
5. Compare document contents.
6. Detect added, removed, and modified text.
7. Render PDFs side-by-side using PDF.js.
8. Provide a Difference Navigation Panel.
9. Clicking a difference should:
   - Navigate to the correct page
   - Scroll both viewers
   - Highlight the changed content
10. Keep the architecture modular.

Out of Scope:
- OCR
- AI summaries
- Standards checking
- Authentication
- Appwrite
- Datadog
- Vector databases

Expected Deliverables:
- Repository structure
- Frontend architecture
- Backend architecture
- API design
- Component hierarchy
- Data models
- Local development instructions
- Step-by-step implementation plan
